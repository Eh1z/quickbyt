# Food Delivery Mobile App

A production-grade food delivery mobile application built to demonstrate real-world mobile engineering, scalable backend design, and polished user experience.

This project is not just a demo, it is a full-stack mobile product designed the way real consumer apps are built, with performance, scalability, and maintainability in mind.

---

## Overview

This app delivers a modern food ordering experience with secure authentication, intelligent discovery, and a smooth checkout flow. It is built to showcase how I approach building business-ready products for startups and founders.

The architecture prioritizes:

- Clean separation of concerns
- Scalable backend design
- Production monitoring and reliability
- Modern UI and performance best practices

---

## Key Features

- Google Authentication with secure session handling
- Smart search with dynamic filtering
- Cart and order flow with persistent state
- Smooth navigation and transitions
- Responsive mobile UI with modern design patterns
- Error tracking and monitoring using Sentry
- Scalable backend with Appwrite for auth, database, and storage

---

## Tech Stack

### Frontend

- React Native
- TypeScript
- NativeWind (Tailwind CSS for React Native)
- Expo Router and modern navigation patterns

### Backend and Infrastructure

- Appwrite (Authentication, Database, Storage)
- Scalable database architecture
- Secure API communication
- File and media handling

### Monitoring and Reliability

- Sentry for error tracking and performance monitoring

---

## Architecture Highlights

- Typed data models across the app using TypeScript
- Centralized state and predictable data flow
- Backend structured for growth beyond MVP scale
- Clean UI components designed for reuse
- Defensive error handling and logging for production readiness

This architecture mirrors what is used in real startups and consumer products, not hackathon prototypes.
